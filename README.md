# Rapidmvc
PHP mvc framework for easy and rapid web development.

```bash
Rapidmvc
│ 
├───app						# Core folder for mvc structure	
│   ├───controller
│   ├───model
│   └───view
│
├───bootstrap				# Include the application autoloader
│ 
├───cache					# Store system and view cache
│   └───blade
│ 
├───config
│ 
├───helpers
│ 
├───public
│   └───assets
│       ├───css
│       ├───images
│       └───js
│ 
├───routes
│ 
├───services
│ 
├───storage
│   └───sessions
│ 
└───vendor
```
