# Rapidmvc
PHP mvc framework for easy and rapid web development.

```bash
Rapidmvc:.
│ 
├───app
│   ├───controller
│   ├───model
│   └───view
│
├───bootstrap
│ 
├───cache
│   └───blade
│ 
├───config
│ 
├───helpers
│ 
├───public
│   └───assets
│       ├───css
│       ├───images
│       └───js
│ 
├───routes
│ 
├───services
│   ├───app
│   └───query
│ 
├───storage
│   └───sessions
│ 
└───vendor
```
